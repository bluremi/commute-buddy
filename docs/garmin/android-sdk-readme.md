# Connect IQ SDK for Android

> Source: https://github.com/garmin/connectiq-android-sdk

## SDK Download

### Maven

```xml
<dependency>
    <groupId>com.garmin.connectiq</groupId>
    <artifactId>ciq-companion-app-sdk</artifactId>
    <version>2.2.0</version>
</dependency>
```

### Gradle

```groovy
implementation("com.garmin.connectiq:ciq-companion-app-sdk:2.2.0@aar")
```

Latest AAR also available on [Maven Central](https://central.sonatype.com/artifact/com.garmin.connectiq/ciq-companion-app-sdk/).

---

## Samples

### Comm Android Sample
- Sample companion application for Connect IQ on Android
- Compile using Android Studio, run on phone

### Comm Watch Sample
- Sample Connect IQ watch app that pairs with the Comm Android app
- Available in the `samples` folder of the [Connect IQ SDK for devices](https://developer.garmin.com/connect-iq/sdk/)
