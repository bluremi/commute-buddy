## Google Play Billing Integration
Implementing the financial model is deferred until after product-market fit is validated with the initial test group.

## Subscription Model
The planned structure is a $2.99/month subscription to cover the ongoing Gemini API and infrastructure costs (enforcing the existing cap of 50 API calls/day per user).

## Trial Period & Abuse Prevention
New users will receive a 7-day free trial. To prevent users from resetting the trial by clearing local app data or reinstalling the app, trial eligibility and subscription state will be strictly enforced server-side via Google Play Billing tied to the user's Google account.
